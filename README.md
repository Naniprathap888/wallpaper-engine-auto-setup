To activate the product key and apply the configuration without a GUI:

```bash
./we-patch --activate --profile="Forest_Night_Ultrawide" --key="XXXX-XXXX-XXXX-XXXX"
```

**Flags Explained:**

| Flag | Type | Purpose |
|------|------|---------|
| `--activate` | Boolean | Initiates the key validation and feature unlocking sequence |
| `--profile` | String | Path to a JSON profile (like the one above) |
| `--key` | String | Your legitimate 16-character product key |
| `--force` | Boolean | Overwrites existing configuration without backup |
| `--dry-run` | Boolean | Simulates activation without making changes |

**Example with Dry Run:**
